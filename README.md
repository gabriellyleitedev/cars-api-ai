# 🚧 Projeto em desenvolvimento

---

## consolidando conhecimentos em Backend moderno:

- **Arquitetura de Pastas:** Organização de código usando o padrão de módulos (Controller, Service, Repository).
- **Segurança e Git:** Uso estratégico do `.gitignore` para proteção de variáveis de ambiente (`.env`).
- **Banco de Dados Relacional:** Modelagem de dados com **TypeScript** e **Drizzle ORM**, conectando a um banco PostgreSQL hospedado no **Neon**.
- **Migrações:** Entendimento do fluxo de versionamento de banco de dados (Generate e Push).
- **Ambiente de Desenvolvimento:** Configuração de scripts, variáveis de ambiente e execução de ferramentas via terminal (`npx`).
